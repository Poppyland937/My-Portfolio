Project: Open Mind Project - Charity Organisation Landing Page

Description:

I had the pleasure of designing and developing a landing page for the Open Mind Project, a fictional charity organisation dedicated to supporting mental health initiatives. The main goals were to create an engaging and user-friendly landing page that highlights the organisationâ€™s events and encourages donations.

Project Images:

Role and Responsibilities:

As the web designer and developer, I was responsible for creating the layout, designing the user interface, and implementing interactive elements. Using HTML and CSS, I crafted a visually appealing and responsive design. I also incorporated a touch of JavaScript to enhance the user experience.

Key Features:

  • Welcoming Design: Developed a warm and inviting design that aligns with the mission of the Open Mind Project
  • Event Section: Created a dedicated section to showcase upcoming events, with detailed descriptions and easy-to-find information.
  • Donation Section: Designed an intuitive and secure donation form to encourage contributions.
  • Responsive Design: Ensured the landing page is fully responsive, providing an optimal viewing experience across all devices.
  • Interactive Elements: Utilised JavaScript to add interactivity, such as event details pop-ups and form validation.

Challenges

One of the main challenges I faced was balancing aesthetics with usability. It was essential to ensure that the design was visually appealing while also being easy to navigate and user-friendly. Creating compelling and informative content was another significant challenge, as it needed to effectively communicate the organisation's mission and motivate visitors to participate in events and make donations. Ensuring the landing page looked great and functioned well on all devices, including desktops, tablets, and smartphones, was crucial for providing a consistent user experience. Lastly, incorporating JavaScript to add interactivity, such as event details pop-ups and form validation, posed a challenge in making the experience engaging without overwhelming users.

Outcome:

The final landing page effectively communicates the mission of the Open Mind Project, making it easy for visitors to learn about events and make donations. This project allowed me to refine my skills in web design and development, particularly in creating user-centric, interactive web pages.
